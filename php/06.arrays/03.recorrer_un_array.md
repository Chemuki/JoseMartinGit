## Recorrer un array

Cuando queremos recorrer todos los elementos de un array, podemos hacerlo con cualquiera de los bucles vistos en temas anteriores:

```php
$array = array ("valor1","valor2","valor3","valor4");
for ($i = 0; $i<count($array); $i++){
  echo $array[$i];
}
```
游늳 **Resultado:**
```php
valor1valor2valor3valor4
```
```php
$i = 0;
while ($i < count ($array)){
  echo $array[$i];
  $i++; 
}
```
游늳 **Resultado:**
```php
valor1valor2valor3valor4
```
```php
$i = 0; 
do{
  echo $array[$i];
  $i++;
}while ($i<count($array));
```
游늳 **Resultado:**
```php
valor1valor2valor3valor4
```

Aparte de los bucles vistos anteriormente, contamos con la expresio패n `foreach`, la cual nos permite iterar un array de una manera muy sencilla.

Una de las ventajas a la hora de recorrer un array con esta estructura es que, por cada iteracio패n del bucle, el valor al que estamos accediendo se almacena en una variable con la cual podemos trabajar.

```php
$array = array ("valor1","valor2","valor3","valor4");
foreach ($array as $valor){
  echo $valor;
}
```
游늳 **Resultado:**
```php
valor1valor2valor3valor4
```

La variable `$valor` se rellenara패 en cada iteracio패n del bucle, tomando en cada una de ellas el contenido de cada una de las posiciones del array.

> 游댠 _Esa variable so패lo esta패 accesible dentro del bucle._

> 游닇 _Si el array que queremos recorrer es asociativo, podemos obtener, en cada vuelta del bucle, el valor correspondiente para la clave y para el valor:_

```php

foreach ($arrayAsoc as $clave => $valor){
  echo $clave." - ".$valor;

```


### Array multidimensionales

En el caso de disponer de un _array multidimensional_ (array cuyos valores son otros arrays diferentes) podemos utilizar la expresio패n list para poder obtener sus elementos. 

```php
$array = [
  [1, 2],
  [3, 4],
];
foreach ($array as list($a, $b)) {
	echo "A: $a; B: $b\n";
}
// En la variable $a se almacena el primer
// valor y en $b el segundo de cada array
```
游늳 **Resultado:**
```php
A: 1; B: 2
A: 3; B: 4
```
